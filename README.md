# Neurodegenerative-Proteomics-Clustering

This repository contains MATLAB code for unsupervised clustering analysis of CSF proteomics data across multiple neurodegenerative diseases.

## Overview

The project investigates molecular clustering patterns across:

- Alzheimer's disease (AD)
- Amyotrophic lateral sclerosis (ALS)
- Behavioural variant frontotemporal dementia (bvFTD)
- Corticobasal syndrome (CBS)
- Primary progressive aphasia (PPA)
- Progressive supranuclear palsy (PSP)
- Healthy controls

Methods include:

- PCA
- k-means clustering
- spectral clustering
- cluster validation
- protein module analysis
- differential protein analysis
